lao version 1.1.0 - 20170904

ao.isBigEndian

Test if this computer uses big-endian byte ordering. 

boolean isBigEndian = ao.isBigEndian()

Return Values

  • isBigEndian
    • True if this computer uses big-endian byte ordering.
    • False if this computer uses little-endian byte ordering.
