ao.appendGlobalOption

Append a key-value pair to the internal linked list of global options.
These options are normally set in /etc/libao.conf or in ~/libao but you may need to over-ride those choices at run-time.
  

boolean rv = ao.appendGlobalOption(string key, string value)

Parameters

  • key
    • A string holding the option key. This can be "default_driver", "quiet", "debug" or "verbose". See the AO Options in xiph.org/ao/doc/config.html
  • value
    • A string holding the option value. For default_driver, this will be one of the drivers returned by driverInfoList() and for the other options it should be either yes or no

Return Value

Returns success or failure.
