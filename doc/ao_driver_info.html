ao.driverInfo

Get information about a particular driver. 

table info = ao.driverInfo(number driverId)

Parameters

  • driverId
    • The ID number of the driver as returned by either ao.driverId or ao.defaultDriverId.

Return Values

  • info
    • If a driver was found with the specified ID, this will be a table containing info about the driver.
    • If no driver was found, this will be nil.
    • See also: info
