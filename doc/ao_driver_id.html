ao.driverId

Looks up the ID number for a driver based upon its short name. The ID number is needed to open the driver or get info on it. 

number driverId = ao.driverId(string shortName)

Parameters

  • shortName
    • The short name of the driver. See xiph's list of drivers.

Return Values

  • driverId
    • If a driver was found with the specified short name, this is the ID of the driver.
    • If no driver was found, this will be nil.
