option

Instead of this being a userdata which is restrictive and cumbersome, you use a regular table with key->value pairs.

You can supply option tables to ao.openLive and ao.openFile.

The keys you use in the table are the options you give to the driver. If you wonder what options you can use, use ao.driverInfo.
