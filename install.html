installation instructions

lao can be installed in seconds with LuaRocks. If you have LuaRocks installed, you can install lao with the following command: 

luarocks install ao

If you have libao installed this should automatically download a stable version of lao from the git repository, build, link, install and all that for you.

To make sure that lao installed successfully, you can download and run the following Lua script: ao_example.lua

If you hear a static boop for about a second, that means everything worked out! Great! Party away, and don't forget the crisps!
