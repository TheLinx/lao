h1. lao

lao is a Lua module that provides bindings to xiph.orgs libao to provide audio output through Lua.

h2. Dependencies

You need libao installed to use lao, and you need the libao headers to build it.

h2. How to Install

Use LuaRocks to install the "ao" package.

h2. Using lao in your script

Require "ao", and you're off! The way you use lao is almost identical to the way you use libao.

For more information, "read the docs!":http://thelinx.github.com/lao
